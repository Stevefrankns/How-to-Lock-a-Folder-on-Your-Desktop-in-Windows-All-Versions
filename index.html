How to Lock a Folder on Your Desktop in Windows (All Versions)

Looking for a way to password protect a specific folder sitting on your Windows desktop? Whether you share a PC at home, need to secure work files from a remote location, or just want basic privacy, getting this right matters.

Quick Answer

Windows does not have a native "right-click and add password" feature for individual desktop folders. To truly lock a desktop folder, you must either use a batch script (low security), Windows Encrypting File System (ties to your user account, not a standalone password), or dedicated third-party software like Folder Lock (high security, standalone password).

Why Locking a Desktop Folder is Different

The desktop is the most convenient place to store files, which makes it the most vulnerable. When you search for "how to lock a folder on desktop windows 11" or "lock folder in windows 7", you're usually looking for a simple password prompt before a folder opens.

Here is the reality: The Windows operating system does not offer simple folder password protection out of the box.

Many guides online will show you how to hide a folder or use a command prompt script to make it disappear. While these tricks work for casual privacy against a curious child, they offer zero actual encryption or security against anyone with basic computer knowledge. If you need to secure financial documents, private photos, or sensitive work files, you need to understand the difference between hiding, permissions, and actual locking.

In this guide, we break down every available method—from free built-in tools to professional software—so you can choose the right level of security for your desktop files.

Method 1: The CMD Batch Script (Free, Low Security)

Difficulty:
Medium Security:
Low

This is the most common free method shared online. It uses a simple text script to create a hidden folder that requires a password to unhide. It works on Windows 10, Windows 11, and older versions like Windows 7.

FolderLock.bat 
cls
@ECHO OFF
title Folder Locker
if EXIST "Control Panel.{21EC2020-3AEA-1069-A2DD-08002B30309D}" goto UNLOCK
if NOT EXIST Private goto MDLOCKER
:CONFIRM
echo Are you sure you want to lock the folder(Y/N)
set/p "cho=>"
if %cho%==Y goto LOCK
if %cho%==y goto LOCK
if %cho%==n goto END
if %cho%==N goto END
echo Invalid choice.
goto CONFIRM
:LOCK
ren Private "Control Panel.{21EC2020-3AEA-1069-A2DD-08002B30309D}"
attrib +h +s "Control Panel.{21EC2020-3AEA-1069-A2DD-08002B30309D}"
echo Folder locked
goto End
:UNLOCK
echo Enter password to unlock folder
set/p "pass=>"
if NOT %pass%== YOUR_PASSWORD_HERE goto FAIL
attrib -h -s "Control Panel.{21EC2020-3AEA-1069-A2DD-08002B30309D}"
ren "Control Panel.{21EC2020-3AEA-1069-A2DD-08002B30309D}" Private
echo Folder Unlocked successfully
goto End
:FAIL
echo Invalid password
goto end
:MDLOCKER
md Private
echo Private created successfully
goto End
:End

How to use it:

  1. Create a new text document on your desktop.
  2. Paste the code above into the document.
  3. Find YOUR_PASSWORD_HERE in the code and replace it with your desired password.
  4. Save the file as FolderLocker.bat (ensure "All Files" is selected in the Save As type dropdown).
  5. Double-click the .bat file. It creates a folder named "Private". Put your files in there.
  6. Double-click the .bat file again to lock and hide the folder.

Limitations: Anyone who right-clicks the .bat file and selects "Edit" can read your password in plain text. This only protects against someone who doesn't know how to use a computer.

Method 2: Windows EFS (Free, Account-Based)

Difficulty:
Easy Security:
Medium (Depends on login)

The Encrypting File System (EFS) is built into Windows Pro, Enterprise, and Education editions (not available on Windows Home). It encrypts the folder, turning the text green.

  1. Right-click the desktop folder and select Properties.
  2. Click the Advanced button under the General tab.
  3. Check Encrypt contents to secure data.
  4. Click OK, then Apply.

Limitations: EFS is tied to your Windows user account. It doesn't ask for a password when you double-click the folder. It simply opens. It only protects the folder if another user logs into the computer with a different account. If you leave your PC logged in and walk away, the folder is completely accessible.

How to Remove Lock from Desktop Folder (Troubleshooting)

If you are struggling to access a folder you locked previously, here are the safest ways to recover access as the owner.

If you used a batch script, simply right-click the .bat file, select "Edit", and look for the line if NOT %pass%== YOUR_PASSWORD goto FAIL. Your password is right there in plain text.

If you used premium software like Folder Lock, check if you enabled Master Password recovery via email during setup. Attempting to bypass professional encryption tools without the password will likely result in data corruption.

If the folder text is green (EFS encryption) and you receive an "Access Denied" error, you are likely logged into a different Windows user account than the one used to encrypt the folder. You must log out and log back in with the original user account, or import the EFS backup certificate you created when you first encrypted the folder.

Major Windows updates occasionally reset folder permissions. If you used third-party software, your locked folders are generally safe within their encrypted containers. If you used native Windows permissions and lost access after an update, you will need to right-click the folder, go to Properties > Security > Advanced, and take ownership of the folder as an administrator.
